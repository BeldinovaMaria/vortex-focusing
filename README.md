# Focusing of circularly polarized vortex beam

MATLAB simulation of focusing through a sector diaphragm.

## Description
This code simulates the focusing of circularly polarized light with an optical vortex (topological charge = 5) through a sector diaphragm aperture.

## Files
- `main.m` - main script to run the simulation
- `circularPar.m` - class for circular polarization parameters
- `RichWolf.m` - base class for Rich-Wolf diffraction theory
- `debary2DGeneral.m` - 2D Debye focusing class
- `modiffigures.m` - figure formatting function
- `modiflabels.m` - label formatting function

## How to run
1. Open MATLAB
2. Run `main.m`

## Parameters
- Wavelength: 0.633 μm
- NA: 0.95
- Vortex charge: 5
- Sector aperture: π/8
